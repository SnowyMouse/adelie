# Adélie

> **Note:** This project does not really do much of anything yet!

Adélie is a Game Boy (DMG) and Game Boy Color (CGB) emulator written in Rust.

## Goals

These are the goals for Adélie:
* Cycle accuracy
* Compatibility with retail games
* C API
